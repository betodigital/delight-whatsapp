== Description ==

O Delight WhatsApp é um plugin simples e eficaz que adiciona um botão flutuante do WhatsApp ao seu site WordPress. Com ele, seus visitantes podem entrar em contato diretamente via WhatsApp com apenas um clique.

### Características principais:

* **Botão flutuante personalizável** - Posicione o botão à esquerda ou direita da tela
* **Posição vertical ajustável** - Defina a distância do botão em relação à parte inferior da tela
* **Mensagem de saudação** - Exiba uma mensagem de boas-vindas opcional
* **Integração com Google Analytics** - Rastreie cliques no botão
* **Integração com Google Tag Manager** - Configure eventos personalizados
* **Design responsivo** - Funciona perfeitamente em dispositivos móveis
* **Fácil configuração** - Interface simples no painel administrativo

### Como usar:

1. Instale e ative o plugin
2. Vá para Configurações > Delight WhatsApp
3. Configure seu número do WhatsApp
4. Personalize a posição e aparência do botão
5. Salve as configurações

O botão aparecerá automaticamente em todas as páginas do seu site.

== Installation ==

### Instalação automática:

1. Faça login no seu painel do WordPress
2. Vá para Plugins > Adicionar novo
3. Pesquise por "Delight WhatsApp"
4. Clique em "Instalar agora" e depois "Ativar"

### Instalação manual:

1. Baixe o arquivo ZIP do plugin
2. Vá para Plugins > Adicionar novo > Enviar plugin
3. Selecione o arquivo ZIP e clique em "Instalar agora"
4. Ative o plugin

### Configuração:

1. Vá para Configurações > Delight WhatsApp
2. Digite seu número do WhatsApp no formato internacional
3. Configure as demais opções conforme desejado
4. Clique em "Salvar alterações"

== Frequently Asked Questions ==

= Como devo formatar o número do WhatsApp? =

Use o formato internacional com código do país, DDD e número. Exemplo: 5511999999999 (Brasil)

= O plugin funciona em dispositivos móveis? =

Sim, o plugin é totalmente responsivo e funciona perfeitamente em smartphones e tablets.

= Posso personalizar a aparência do botão? =

Atualmente, você pode personalizar a posição do botão. Personalizações visuais mais avançadas podem ser feitas via CSS.

= O plugin afeta a velocidade do site? =

Não, o plugin é otimizado e adiciona apenas os recursos necessários, mantendo o impacto mínimo na performance.

= Como rastrear cliques no botão? =

Configure o Google Analytics ou Google Tag Manager nas configurações do plugin para rastrear automaticamente os cliques.
